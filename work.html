myFlix (frontend)

The myFlix frontend is a React-based web application that allows users to sign-up and log-in the app, browse and search through a list of movies, view detailed information about the movies, and manage their profile/account information and favorite movies, all while being authenticated via a token-based login system. The project is hosted on Netlify.

myFlix (backend)

This code sets up a RESTful API for my movie app myFlix using Express, MongoDB, and Mongoose. The API allows users to sign-up, manage their profiles, update their favorite movies list, and access movie details by title, genre, or director. It includes authentication with JWT, validation for user input, and CORS settings to control access from specific origins. The app also serves static files and handles errors gracefully.
